# ty and Zed Type Checking Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add reproducible ty type checking and make ty the primary Python language server in Zed.

**Architecture:** Pin ty in Caption's uv development environment and use `ty check` as a project gate. Configure Zed at the repository level to run ty for type analysis and Ruff for linting, import sorting, and formatting.

**Tech Stack:** Python 3.13, uv, ty, Ruff, Zed, pytest
